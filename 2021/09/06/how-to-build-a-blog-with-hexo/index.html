如何通过hexo搭建博客

准备

下载 git 和 Node.js
  1. git 下载 直接前往官网下载 https://git-scm.com/downloads
  2. Node.js 下载 直接前往官网下载 https://nodejs.org/en/[^nodejs下载说明]

[^nodejs下载说明]: LTS 为稳定版，Current为最新版，推荐下载 LTS 版

检查下载是否成功
  1. win+R 打开运行 输入cmd 调出命令行
  2. 输入 git –version 若出现git 版本信息，则git下载完成

  1. 输入 node -v 若出现Node.js 版本信息，则Node.js下载完成

安装hexo

创建文件夹
  1. 在想要存放博客的地方创建一个用于存放博客的文件夹，如blog 文件夹
安装hexo

  1. 打开blog 文件夹，右键进入git bash here ，进入 Git Bash 界面

  2. 输入 npm install -g hexo-cli ，下载 hexo ，

  3. 输入 hexo -v ，若出现hexo 版本信息，则安装成功

  4. 完成后输入hexo init ，进行初始化

  5. 此时文件夹中有如下文件：

    • node_modules ：依赖包
    • public ：存放生成的页面
    • scaffolds ：生成文章的模板
    • source ：用来存放你的文章
    • themes ：博客主体
    • _config.yml ：博客的配置文件

  6. 输入 hexo g && hexo s (hexo server) ，打开 hexo 服务

  1. 在浏览器输入localhost:4000就可以看到你生成的博客了，大概如下

GitHub创建个人仓库

注册GitHub
创建个人仓库
  1. 登录 GitHub 后，点击New repository ,新建仓库
  2. 创建一个名为 “你的用户名.github.io” 的仓库
  3. 点击 Create repository ，创建完成

生成SSH添加到GitHub

  1. 再次进入 git bash中，输入 git config –global user.name “yourname” 和 git config –global user.email “youremail” ，引号中分别输入你的 GitHub 用户名和你的 GitHub 邮箱
  2. 可以输入 git config users.name 和 git config user.email 检查输入是否正确
  3. 此时 生成了一个 .ssh 文件夹， 并呈现了它的路径
  4. 找到其文件夹，其中 id_rsa 是你这台电脑的私人密钥， id_rsa.pub 是公共密钥
  5. 打开你的GitHub ，打开 Personal settings 中 SSH and GPG keys 中的 New SSH key
  6. title可以随便输入，在 key 中复制之前 id_rsa.pub 的所有内容
  7. 点击 Add SSH key ,完成

将hexo部署到GitHub

  1. 进入你创建的 blog 文件夹，打开站点配置文件 _config.yml ，翻到最后，修改为

其中，yourgithubname为你的GitHub账户名

  1. 安装deploy-git，输入npm install hexo-deployer-git –save

  1. 部署到GitHub，输入 hexo clean && hexo g && hexo d

  1. 然后就可以在 http://yourgithubname.github.io 上看见你的博客啦

创建博客文章

  1. 进入 git bash 中，输入 hexo new 文件名 ，（例如：创建一个名为newblog的博客）

注：文件名最好不要用中文

  1. 在source文件夹中就可以看到你新创建的.md文件

  2. 编辑完成后，输入 hexo clean && hexo g && hexo d 即可完成
